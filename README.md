# QuickPing(快捷宣告)![](docs/images/ping.png)
###### *灵感来源于饥荒同名模组*

## 概述
QuickPing是一个Minecraft模组，它为玩家提供了一个轮盘来声明自己的位置,标记一处地点，高亮生物等等功能。

![](docs/images/radial.png)


## 主要功能
- 标记地点

    按下g键可以快捷标记,或者按下alt调出轮盘，选择Ping。

<img height="400" src="docs/images/diamond!.png" width="600"/>


- 高亮生物

    对着生物使用标记可以高亮该生物，支持自定义颜色

  <img height="512" src="docs/images/bad_ping.png" width="512"/>


- 快捷宣告

    目前支持宣告手持物品，宣告当前坐标，宣告发现的方块。
- 
## 模组配置
- 颜色配置：该颜色控制高亮生物的颜色，粒子颜色，快捷声明时文字的颜色。
- 光束配置：
    - 距离表示渲染光束的最近距离，低于该距离光束将不会渲染。(0~100),默认值为50
    - 光束时长表示光束的持续时间(0~10000),默认值为200
    - 高度代表光束的高度(0~100),默认值为50
